# C - Function pointers

This project contains C programs that demonstrate the use of function pointers.

## Learning Objectives
- What are function pointers and how to use them
- What does a function pointer exactly hold
- Where does a function pointer point to in the virtual memory

## Tasks

0. **0-print_name.c** - Function that prints a name using a function pointer
1. **1-array_iterator.c** - Function that executes a function on each element of an array
2. **2-int_index.c** - Function that searches for an integer using a comparison function
3. **3-main.c, 3-op_functions.c, 3-get_op_func.c, 3-calc.h** - Calculator program using function pointers
